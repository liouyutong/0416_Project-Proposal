# 🦷 牙視界 (OralVision)：居家 AI 口腔影像分析系統
> 應用於智慧醫療物聯網的精準口腔健康監測解決方案

## 📖 專案簡介
本專案為第 9 組的成果，開發一套結合 **AI 影像辨識**、**SLM/SLAM 拼接技術**與 **LLM+RAG 智慧導航**的口腔健康監測系統。旨在填補民眾每半年洗牙間的 180 天監控真空期，將醫學範式從「反應性治療」轉向「精準預防型監管」。

## 🚀 核心功能 (Key Features)
*   **🤖 AI 多維度病灶識別**：自動偵測齲齒 (Caries)、牙結石 (Calculus)、牙齦萎縮及口腔潰瘍。
*   **📸 智慧引導影像採集**：透過動態引導框，協助非專業用戶採集高品質、高重疊率的影像。
*   **🧩 全景影像拼接**：利用 SIFT/ORB 與 SLAM 演算法，將口內照片縫合為完整的口腔全景地圖。
*   **⚠️ 高風險預警機制**：針對疑似惡性病變（破、斑、硬、突、腫）強制發出緊急就醫導引。
*   **💡 LLM+RAG 個人化保健**：結合醫學知識庫，根據個人病灶生成量身定制的潔牙指引（如：針對智齒區的貝氏刷牙法）。
